================================================================================
TROUBLESHOOTING
================================================================================

If npm install fails:
npm cache clean --force
rm -r node_modules
npm install

If build fails:
npm run build

If git push fails - authentication error:
1. Get Personal Access Token from GitHub
2. Settings â†’ Developer settings â†’ Personal access tokens â†’ Generate new token
3. Select "repo" scope
4. Copy token
5. When prompted for password during git push, paste the token

If node version is wrong:
nvm install 20.11.1
nvm use 20.11.1

If git remote is wrong:
git remote remove origin
git remote add origin https://github.com/YOUR_USERNAME/harvest-basket.git
